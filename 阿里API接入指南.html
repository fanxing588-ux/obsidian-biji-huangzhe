笔记拾荒者

写笔记时,自动从你过去写过的笔记里捞相关的,浮现在右边。
3 步上手,大概 10 分钟。

1装插件

把作者给你的 笔记拾荒者-v0.1.zip 解压,会得到一个 biji-huangzhe 文件夹和这份说明。

打开 Obsidian → 左下角齿轮(设置)→ 左栏 第三方插件:

  1. 把"安全模式"关掉(右侧 toggle)
  2. 下方"已安装的插件" 标题右边有个文件夹小图标 📁 → 点它,会弹出 .obsidian/plugins/ 这个目录
  3. 把刚才解压出的 biji-huangzhe 文件夹整个 拖进去
  4. 回到 Obsidian → 设置 → 第三方插件 → 右下角点 重新载入(或者关掉 Obsidian 再开)
  5. 在"已安装的插件"列表里找到 笔记拾荒者 → 把它右边的 toggle 打开
如果你看不到那个文件夹小图标:手动找到 vault 目录(你存笔记的文件夹),进去找到隐藏的 .obsidian/plugins/ 文件夹,没有就自己建一个,然后把 biji-huangzhe 拖进去。

2拿一个免费的 API Key

  1. 打开 https://bailian.console.aliyun.com/
  2. 用支付宝 / 阿里云账号登录,做一下实名
  3. 右上角头像 → API-KEY → 创建 → 复制 sk-xxxxxx
免费的,新用户送 100 万 token,够用半年到一年。不绑卡也能用。

3填进插件 + 索引

回到 Obsidian → 设置 → 笔记拾荒者,顶部填 4 个字段:

API Endpointhttps://dashscope.aliyuncs.com/compatible-mode/v1
Modeltext-embedding-v4
Dimensions2048
API Key刚才那个 sk-xxxxxx

填完往下拉,点 开始索引。等几分钟跑完(几百篇笔记 5 分钟内)。

✓ 完事

随便打开一个笔记开始写,停 2 秒,右边自动浮现卡片。卡片底部 看原文 → 一点就跳到原段落。

不想被打扰:Ctrl+P 搜 切换专注模式,一键暂停 / 恢复。

有问题随时找作者,贴一下 Obsidian 控制台(Ctrl+Shift+I → Console)截图就行。
