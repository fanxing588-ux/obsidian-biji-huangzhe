安装 + 以后怎么更新

第一次装 大概 3 分钟

1
打开 Obsidian，左下角点 ⚙️ 设置
2
点 第三方插件 → 关闭"安全模式"
3
点 浏览 → 搜 BRAT → 安装 → 启用
4
左边菜单往下拖，找到 BRAT 点开 → Add Beta plugin
5
粘这一行，点 Add Plugin：
fanxing588-ux/obsidian-biji-huangzhe
6
回到 第三方插件，打开「笔记拾荒者」开关
✓ 完成，可以用了

以后更新 不用操心

右下角弹通知 → 点 Update
就这样。BRAT 每小时自动检查一次。
实在等不及：设置 → BRAT → Check for updates to all beta plugins

出问题了 三种情况

  • 装完没反应：按 Ctrl + R 重新加载 Obsidian
  • 找不到 BRAT：左边菜单可以拖动，往下找"社区插件"分类
  • 报 Network error：你网络访问不到 GitHub，挂代理或换网试试
