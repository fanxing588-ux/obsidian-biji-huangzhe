笔记拾荒者 · 更新手册

小白版 · 三分钟看完，永远不用愁怎么更新
一句话总结：装好之后，更新这件事基本不用你操心 —— BRAT 会自动提示你，你点一下"Update"就行。

方式一：等通知（最省事）

BRAT 默认每小时检查一次更新。当作者发布新版本，只要你的 Obsidian 是开着的，右下角会弹出一个通知，告诉你"笔记拾荒者 有新版本可用"。

你只需要做一件事：点通知里的 Update 按钮。装好后 BRAT 会再弹一个提示，告诉你要不要立即重启 Obsidian 来生效 —— 选 Yes 就行。

方式二：手动检查（不想等就用这个）

如果你听说了新版发布，但 BRAT 还没自动检测到，手动来一下：

  1. 左下角齿轮图标打开 设置
  2. 左侧菜单往下找 BRAT，点进去
  3. 找到 Check for updates to all beta plugins 按钮，点它
  4. BRAT 会立刻检查所有 beta 插件。如果"笔记拾荒者"有新版，会自动装好
  5. 右上角会弹"Plugin updated"提示。按 Ctrl + R 重载 Obsidian 让新版生效

方式三：实在不行，手动覆盖

很罕见的情况：你的网络访问不到 GitHub（公司内网 / 国内某些地区） —— BRAT 会反复报 Network error。这时候只能手动来：

  1. 用能访问 GitHub 的网络（或代理）打开：
    https://github.com/fanxing588-ux/obsidian-biji-huangzhe/releases
  2. 找到最新的 Release（页面最上面那个），下载 三个文件：
    • main.js
    • manifest.json
    • styles.css
  3. 找到你的 vault 文件夹，进入：
    你的Vault/.obsidian/plugins/biji-huangzhe/
    （.obsidian 是隐藏文件夹，看不到的话开启"显示隐藏文件"）
  4. 把刚下的三个文件覆盖这个文件夹里的同名文件
  5. 回到 Obsidian，按 Ctrl + R 重载

检查当前版本

想知道自己现在装的是哪个版本？

设置 → 第三方插件 → 找到「笔记拾荒者」→ 看插件名下面写的 Version: 0.x.x

然后跟 GitHub 上的 Releases 页面最新版本对比一下，就知道要不要更新。

常见问题

更新完之后插件不工作 / 出错了

  1. 按 Ctrl + R 重载 Obsidian（最常见的解决办法）
  2. 还不行：设置 → 第三方插件 → 把「笔记拾荒者」的开关关掉再打开
  3. 还不行：去 GitHub Issues 反馈，附上插件版本号 + 报错截图

BRAT 提示 "Could not find a valid release"

说明 BRAT 在 GitHub 仓库里没找到符合规范的 Release。一般是临时网络问题，等几分钟再点 Check for updates 即可。

不想自动更新，想自己控制节奏

BRAT 设置 → 关闭 Auto-update plugins at startup。之后只有你手动点 Check for updates 才会检查。
